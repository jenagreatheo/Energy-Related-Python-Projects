# Energy-Related-Python-Projects
1. Live Solar Grid Simulator: Renewable energy (sunlight) micro-grid simulation tracking solar power generation against variable hourly electrical demands using both live API feeds and manual inputs.
2. Smart Solar Power Grid Controller: A MicroPython-based smart solar grid simulator on the Raspberry Pi Pico that estimates solar power generation and demonstrates automated energy management using LEDs and a buzzer (can be simulated on Wokwi).
